# Songsheet Reader

A single-file HTML app for reading chord/lyric sheets, with clickable fretboard diagrams, transpose, and auto-scroll.

## Usage

Here is the live link, check it out: [https://sinan-ozel.github.io/songsheet-reader/](https://sinan-ozel.github.io/songsheet-reader/)

Paste your chords-over-lyrics text into the left panel:

```
[Verse 1]
Am              G
Some lyric line goes here
```

Chords render highlighted, with diagrams collected in the glossary on the right.



### Local Usage

Open `index.html` in any browser. No build step, no server, no dependencies to install.

## Features

- Click any chord to highlight it and its diagram
- Transpose up/down
- Adjustable text size
- Auto-scroll with speed control
- Manual prev/next line navigation
